# Data licensing and attribution

The code in this repository is released under the MIT License. The raw data files in `data/krakow_pb/` are Pabulib `.pb` files and are included for replication of the empirical results in the paper.

This repository does **not** assign a new license to the Pabulib data. Users of the data should follow Pabulib's own terms, citation policy, and any restrictions attached to the original municipal data.

Please cite Pabulib when using the data. The relevant references are included in `CITATION.bib`:

- Faliszewski et al., *Participatory Budgeting: Data, Tools and Analysis*, IJCAI 2023.
- Stolicki, Szufa, and Talmon, *Pabulib: A Participatory Budgeting Library*, arXiv 2020.

Pabulib website: <https://pabulib.org/>
